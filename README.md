## Installation

### Clone

- Clone this repo to your local machine using `https://github.com/dennisagarwal/movie-weather-app.git`

### Setup

 **1.** Navigate to the client folder in your terminal and install npm like this:

```
$ npm install
```
### Launch application

```
$ npm start
```
